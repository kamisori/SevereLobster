Finden Sie alle Sterne in den tiefen der Galaxis Vom Universum hat vermutlich schon jeder gehört, aber das weite All durch Geschicklichkeit und vor allem Schnelligkeit zu erkunden ist erst mit diesem Spiel möglich.

Ziel des Spiels:

 Alle Sterne in Sternenhimmel DELUXE in kürzester Zeit aufdecken, mit möglichst wenig Schritten versteht sich.

Vorbereitung:

 Java Runtime Enviroment muss auf dem Rechner installiert sein. Verfügbar unter http://www.java.com

Spielmodi:

 Ein einfaches Spiel kann über das Hauptmenü mit Klick auf "Neues Spiel starten" gestartet werden.
"Eigenes Spiel starten", um ein lokales Spiel auszuwählen. "Online Archiv durchsuchen", um weitere Spiele aus dem Internet zu laden.

Die Herausforderung ist die Kampagne, in dem mit jedem gelösten Sternenhimmel, der nächste freigeschaltet wird. In der Kampagne werden die Sternenhimmel zunehmend schwerer zu lösen sein.
Viel Erfolg!

Erster Blick auf das Spielfeld:

 Die Pfeile zeigen an, in welcher Himmelsrichtung sich Sterne befinden. Die Ziffern geben die Anzahl der Sterne der jeweiligen Spalte bzw. Zeile an.

Spielen:

 Ein Linksklick mit der Maus auf ein Feld, setzt einen Stern, ein Rechtsklick einen Ausschluss.
Durch einen erneuten Klick, wird die getroffene Auswahl rückgängig gemacht. Doch Vorsicht, jeder Klick wird als weiterer Lösungsschritt gezählt und vermindert die erreichbare Punktzahl.
Ebenso wird bei der Endpunktezahl die Zeit beachtet. Je schneller ein Sternenhimmel aufgedeckt ist, desto mehr Punkte gibt es.

Über die Menüleiste kann ein Spiel jederzeit unterbrochen und gespeichert werden, das Spiel eignet sich also perfekt für den Büroalltag.

Während des Spiels kann mit dem Button "Zurück zum Fehler", bis zu dem Punkt zurückgesprungen werden, bei dem alle davor gesetzten Sterne und Ausschlüsse korrekt waren. Allerdings wird jedes zurückgesetzte Feld als weiterer Lösungsschritt gezählt.

Spiel speichern:

 Jederzeit kann ein laufendes Spiel gespeichert werden. Dazu das Menü "Spielen - Spiel speichern" wählen. Ist der aktuelle Spielfortschritt noch nicht gespeichert worden, öffnet sich der Dialog "Spiel speichern unter...". Beim nächsten Speichervorgang wird die bereits angelegte Datei überschrieben. Soll die angelegte Datei nicht überschrieben werden, wird im Menü "Spielen - Spiel speichern unter..." gewählt.

Spiel fortsetzen:

 Um ein gespeichertes Spiel fortzusetzen, wird im Menü "Spielen - Spiel fortsetzen" gewählt. Mit dem Dialog "Spiel laden" wird die Datei mit dem Spielstand gewählt und geladen. Die zeitliche Unterbrechung des Spiels wird ebenfalls berücksichtigt und verlängert nicht die benötigte Lösungszeit.

Zurück zum Fehler:

 Beim Drücken des Buttons "Zurück zum Fehler" werden soweit alle Sterne oder Ausschlüsse wieder entfernt, bis der letzte korrekte Zustand erreicht ist. Hierbei wird jedes zurückgesetzte Feld als weiterer Lösungsschritt gezählt.

Tracking-Modus:

 Der Tracking-Modus wird mit dem Button, der sich rechts vom Spielfeld befindet ein- oder ausgeschaltet. Ist der Tracking-Modus aktiv, kann jederzeit ein Trackingpunkt gesetzt werden, zu denen nacheinander wieder zurückgesprungen werden kann. Hierbei wird jedes zurückgesetzte Feld als weiterer Lösungsschritt gezählt. Die Anzahl der Trackingpunkte wird durch die unterhalb der Trackingsbuttons mit Fähnchen angezeigt

Die Funktion "Zurück zum Fehler" wird durch Aktivierung des Tracking-Modus ausgeblendet. Jederzeit kann zur anderen Option gewechselt werden.

Eigenes Puzzle erstellen:

 Über das Menü "Editieren - Puzzle erstellen" öffnet sich ein Dialog zur Eingabe der gewünschten Spielfeldgröße, die eingegeben wird. Das Spielfeld öffnet sich im Editier-Modus. Sterne und Pfeile werden mit Rechtsklick auf das jeweilige Feld gesetzt. Die Anzahl der Sterne in den Spalten bzw. Zeilen aktualisieren automatisch.

Während des Erstellen des Spiels kann jederzeit überprüft werden, ob das Spiel lösbar ist. Dazu im Menü "Editieren" den Eintrag "Lösungsweg prüfen" auswählen. Um das erstellte Spiel spielen zu können, muss dieses über das Menü "Editieren - Puzzle freigeben" freigegeben werden. Vor der Freigabe des Puzzles wird der Lösungsweg ebenfalls überprüft.

Puzzle speichern:

 Über das Menü "Editieren - Puzzle speichern unter..." bzw. " - Puzzle speichern" wird das Puzzle gepseichert. Mit "...speichern unter..." ist immer ein Puzzlename anzugeben. Ist das Puzzle bereits gespeichert, wird mit "Puzzle speichern" die Datei überschrieben.

Puzzle laden:

 Über das Menü "Editieren - Puzzle laden" wird ein Spiel geladen und in den Editier-Modus gesetzt und kann bearbeitet werden.

Sprache ändern:

 Die Sprache lässt sich über das Menü "Optionen - Sprache ändern" auf Deutsch bzw. Englisch einstellen. Bei Bestätigung der Sprachänderung folgt die Nachfrage ob Sternenhimmel DELUXE neugstartet werden soll. Vorher sollte der Spielfortschritt gespeichert werden, damit kein Spielfortschritt verloren geht.

Avatar ändern:

 Der verwendete Avatar lässt sich über das Menü "Optionen - Avatar ändern" einstellen. Wird ein Spiel gespeichert bleibt dieser für dieses Spiel erhalten.
