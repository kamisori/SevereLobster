Spiel speichern

Spielen ››

Jederzeit kann ein laufendes Spiel gespeichert werden. Dazu das Menü "Spielen - Spiel speichern" wählen. Ist der aktuelle Spielfortschritt noch nicht gespeichert worden, öffnet sich der Dialog "Spiel speichern unter...". Beim nächsten Speichervorgang wird die bereits angelegte Datei überschrieben. Soll die angelegte Datei nicht überschrieben werden, wird im Menü "Spielen - Spiel speichern unter..." gewählt.
